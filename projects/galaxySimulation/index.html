Creating Galaxies in Virtual Reality

About The Project

This project is a Virtual Reality Application that simulates black holes, star movement, and galaxy collisions. The project has five preset galaxy simulations the user can select to view. In each simulation, the user can use Virtual Reality hand controllers to insert a black hole or draw a trail of stars. They can then watch how the newly added objects change the simulation to create new galaxies. The overall goal of the project is to provide a visual experience that will teach people how stars and black holes interact with each other based on Newton’s Laws of Gravity. The project was created using Unity and uses the Oculus Rift Headset and Controllers.
